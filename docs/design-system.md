## Design System (Tokens)

### Colors
- Primary: #4F46E5 (Indigo – main actions, buttons)
- Primary Hover: #4338CA

- Secondary: #E5E7EB (Light gray – backgrounds, borders)

- Success: #22C55E (Finished books)
- Warning: #F59E0B (In progress)
- Neutral: #9CA3AF (Not started)

- Text Primary: #111827
- Text Secondary: #6B7280

- Background: #FFFFFF
- Surface: #F9FAFB

---

### Typography

- Font Family:
  - Primary: Inter, system-ui, sans-serif

- Heading:
  - Size: 20–24px
  - Weight: 600–700

- Body:
  - Size: 14–16px
  - Weight: 400

- Small:
  - Size: 12–13px
  - Weight: 400
  - Use: metadata (author, status)

---

### Spacing
- Base unit: 8px
- Scale: 4 / 8 / 12 / 16 / 24 / 32

---

### Border Radius
- Small: 6px
- Medium: 10px
- Large: 14px

---

### Shadows
- Card: subtle shadow (e.g., 0 2px 8px rgba(0,0,0,0.05))
- Modal: stronger shadow (e.g., 0 10px 30px rgba(0,0,0,0.1))

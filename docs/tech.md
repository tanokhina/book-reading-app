## Tech Stack

- Framework: React 19 (functional components, hooks)
- Build Tool: Vite 7
- Styling: Custom CSS (no CSS frameworks)
- Deployment: Netlify

## Development Notes
- Use modern React patterns (hooks, no class components)
- Keep styling simple and modular
- Optimize for fast load and small bundle size

## Code Guidelines
- Keep components small and reusable
- Use clear naming conventions
- Avoid over-engineering
